Git init: Comando para iniciar o git, criando a pasta .git na pasta do projeto Git add . ou nome da pasta: Comando para passar de working directory para unstaged e depois realizar um commit Git commit -m "msg": Comando para realizar um commit passando uma mensagem Git log: Comando para ver todo o historico de commits com informações de data, author... Git log -online: Comando para ver o historico de commits mais resumidos. Git log n "algum numero": feito para ver os ultimos commits limitando-se pelo numero escolhido Git Diff: Te ajuda a comparar/ver as modifiicações que você fez recente, com o ultimo commit que foi feito. Git Restore: Restaura todas as modificações em andamento de working directory para antes. Git Restore --stage: Comando feito para restaurar todas as mdoficações em andamento em unstaged. Git Reset --soft HEAD~1: Comando feito para desfazer o ultimo commit, e voltar para parte de staged Git push: Comando feito para empurrar/enviar os arquivos do editor de código para a branch no github.
